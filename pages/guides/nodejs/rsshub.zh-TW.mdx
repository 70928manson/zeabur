# 自建 RSSHub

使用 Zeabur 自建 RSSHub。

### 第一步：Fork RSSHub 倉庫

打開 RSSHub 的[官方倉庫](https://github.com/DIYgod/RSSHub)。

點擊 Fork 來 Fork 倉庫到你的個人賬號下。

### 第二步：部署 RSSHub

點擊 **Deploy service** 或 **Add new service** 按鈕，然後選擇 **Deploy your source code**。

![deploy](/framework/deploy.png)

找到你的 RSSHub 倉庫，點擊 import，你的自建 RSSHub 就會開始部署。
