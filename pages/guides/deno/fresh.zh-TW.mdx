# 部署 Fresh 專案

[Fresh](https://fresh.deno.dev/) 是一個現代全端網頁框架，適用於 JavaScript 和 TypeScript。

本文將教您如何在 Zeabur 一鍵部署 Fresh 應用程式。

## 建立 Fresh 應用程式 (可選)

- 請先在您的電腦上安裝 Deno。

- 您可以在 [Deno 官方文件](https://deno.land/manual/getting_started/installation) 中找到更詳細的說明。

利用此指令初始化一個Deno專案並且使用Fresh框架。

```bash
deno run -A -r https://fresh.deno.dev my-project
```

在 `main.ts` 中，將 `port: Deno.env.get("PORT")` 添加到 `start` 函數中。

```typescript
// in main.ts
await start(manifest, { plugins: [twindPlugin(twindConfig)], port: Deno.env.get("PORT")}, );
```

## 開發專案

執行 `deno task start` 以開始開發服務器。

```bash
deno task start
```

接下來，您就可以開始修改程式碼來開發您的 Fresh 專案了。

當您完成時，為其初始化一個 GitHub 儲存庫，並上傳到您的 GitHub。

## 部署專案

在您的專案中，點擊 **部署新服務**  按鈕，並選擇 **部署你的程式碼**。

![deploy](/framework/deploy.zh-TW.png)

搜尋想要部署的 Fresh 程式碼儲存庫，並點擊導入，您的 Fresh 應用就將自動開始部署。

部署完成後，您可以參考 [綁定域名](../../deploy/domain-binding) 文件瞭解如何為您的 Fresh 專案綁定一個能夠讓外部訪問的公開域名。
