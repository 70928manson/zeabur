# 快速開始

從零開始將您的服務部署到 Zeabur 上，只需要以下三個步驟：

### 第一步：註冊帳號

前往 [登入頁面](https://dash.zeabur.com/login) 並且使用 GitHub 帳號進行註冊或登入。

![sign in](/sign-in-screenshot.png)

### 第二步：建立新專案

展開左上角的專案選單，並點擊建立專案按鈕。

![new project](/create-project-screenshot.png)

### 第三步：選擇要部署的服務

點擊畫面右上方的「部署服務」，並且選擇您想要部署的服務。

![deploy service](/deploy-service-screenshot.png)

在部署服務頁面中，選擇左邊的按鈕可以從 GitHub 直接將您的程式碼自動構建並部署上線。

選擇右邊的按鈕則可以從常見的服務（例如資料庫或開源的工具）中選擇一個服務，Zeabur 會自動為您建立一個服務實例。

### 下一步

完成部署以後，您可以了解 [域名綁定](/zh-TW/deploy/domain-binding) 來使用自動帶有 TLS 證書的域名對外暴露您的服務，或是了解 [環境變數](/zh-TW/environment/variables) 來了解如何讓 Zeabur 自動注入您的程式運行所需要的變數。
