# 自訂根目錄

對於某些 Git 儲存庫，例如同時具有前後端的專案，您需要手動為其指明專案根目錄。

### 第一步：部署服務

參照 [部署服務](/zh-TW/get-started) 來部署你的應用

### 第二步：更改專案根目錄

點擊部署好的服務，並進入設定頁面，下拉至 **Root Directory** 一欄

![setting](/deploy/root-directory/setting.png)

輸入想要指定的根目錄，並點擊儲存，儲存成功後，點擊 **Redeploy Service** 重新部署您的服務，服務將會根據指定的根目錄重新建置執行。

![save](/deploy/root-directory/save.png)
