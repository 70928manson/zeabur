# 多环境

在一个工程中，往往需要多环境的隔离，例如生产环境和开发环境。

Zeabur 就为您提供了这样的能力。

### 第一步：创建新环境

点击左上角的新建环境按钮，创建一个新的环境。

![create](/environment/create-env.png)

创建完毕后，您在当前项目中的所有服务会自动进行一次拷贝，在新环境中重新构建一次。

### 第二步：绑定不同分支

对于部署的源代码，您可以在不同环境为同一个服务绑定不同的分支。

点击服务，进入设置页面，下拉至 **Switch Branch** ，选择您想在该环境为其绑定的分支。

例如，为服务选择了 dev 分支，则服务的状态与 dev 分支绑定，您可以为其再起一个 dev 环境的域名，就能够实时查看 dev 环境的线上状态。

![create](/environment/branch.png)
