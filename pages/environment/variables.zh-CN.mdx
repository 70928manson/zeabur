# 添加变量

Zeabur 中可以添加变量，并自动注入到该服务中。

点选该服务并到 **变量** 页面，裡面的 **添加变量** 即可新增变量。

![添加变量](/environment/add-variables.zh-CN.png)

Zeabur 预设会自动注入以下变量：

- `<YOUR_SERVICE_NAME>_HOST`：该服务对内沟通的 host 名称

- `PORT`：该服务对外的 port

- 其他相关服务，例如数据库([PostgreSQL](/databases/postgresql))、[Redis](/databases/redis) 等，也会自动注入相关变量。

这时我们就不需要再手动添加变量到自己的服务中。

![数据库变量](/environment/db-env.zh-CN.png)
