# 服务用量

Zeabur 遵循按量计费的模式，也为您提供了详尽的图表来展示您的服务用量。

点击想查看的服务，并选择 **Metrics** 页，您将会看到真实详尽的用量图表。

### 内存用量

展示服务运行所消耗的内存

![memory](/manage/metric/memory.png)

### CPU 用量

展示服务运行所调用的 CPU
