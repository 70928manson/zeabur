# 快速開始

從零開始將你的服務部署到 Zeabur 上，只需要以下三個步驟：

### 第一步：註冊帳號

前往 [登入頁面](https://dash.zeabur.com/login) 並且使用 GitHub 帳號進行註冊或登入。

![sign in](/sign-in-screenshot.png)

### 第二步：建立新專案

點選建立專案按鈕以建立一個新專案。

![new project](/create-project-screenshot.png)

### 第三步：選擇要部署的服務

點選剛剛建立好的專案，然後再點選「Add Service」按鈕，選擇你想要部署的服務。

![deploy service](/deploy-service-screenshot.png)

在部署服務頁面中，選擇左邊的按鈕可以從 GitHub 直接將你的程式碼自動建置並部署上線。

選擇右邊的按鈕則可以從市集中（例如資料庫或開源的工具）中選擇一個服務，Zeabur 會自動為你建立一個服務實例。

### 下一步

完成部署以後，你可以了解 [網域綁定](/zh-TW/deploy/domain-binding) 來使用自動帶有 TLS 憑證的網域對外暴露你的服務，或是了解 [環境變數](/zh-TW/environment/variables) 來了解如何讓 Zeabur 自動注入你的程式運作所需要的變數。
