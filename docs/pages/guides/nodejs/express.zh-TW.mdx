# 部署 Express 應用

[Express](https://expressjs.com/) 是一個一個輕便靈活的 Node.js 網絡應用框架，為網絡和移動應用提供了一套強大的功能。

在 Zeabur 上一鍵部署你的 Express 應用。

### 第一步：創建 Express 應用（可選）

- 電腦上安裝了 Node.js
- 使用 express-generator 或手動創建

```bash
# 使用 express-generator
npx express-generator
```

你可以在 Express 的[官方文檔](https://expressjs.com/en/starter/installing.html)看到更詳細的教學。

創建完畢後，為其初始化一個 Git 倉庫並部署到你的 Github 上。

### 第二步：部署

在你的項目中，點擊 **Add new service** 按鈕，並選擇 **Deploy your source code**.

![deploy](/framework/deploy.png)

搜索想要部署的 Express 應用倉庫，並點擊導入，你的 Express 應用就將自動開始部署。
