# 部署 Svelte Kit 项目

在 Zeabur 上部署你的 Svelte Kit 项目。

### 第一步: 建立 Svelte Kit 项目（可选）

打开终端，输入

```bash
npm create svelte@latest
```

接下来，你需要安装依赖：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm install
    ```
  </Tab>
  <Tab>
    ```bash
    npm install
    ```
  </Tab>
  <Tab>
    ```bash
    yarn
    ```
  </Tab>
</Tabs>

创建完成后，为其初始化一个 Git 仓库并部署到你的 GitHub 上。

### 第二步：设置 `@sveltejs/adapter-node`

接下来，你需要安装 `@sveltejs/adapter-node`：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add -D @sveltejs/adapter-node
    ```
  </Tab>
  <Tab>
    ```bash
    npm install -D @sveltejs/adapter-node
    ```
  </Tab>
  <Tab>
    ```bash
    yarn add -D @sveltejs/adapter-node
    ```
  </Tab>
</Tabs>

然后，打开 `svelte.config.js`，并将其内容改为：

```js
import adapter from '@sveltejs/adapter-node'; // <-- 注意这里

export default {
  kit: {
    adapter: adapter({ out: 'build' })
  }
}
```

### 第三步：设置 `package.json`

打开 `package.json`，在 `scripts` 中加入一个 `start` 命令：

```json
{
  "scripts": {
    "start": "node ./build/index.js"
  }
}
```

### 第四步：部署 Svelte Kit 项目

在你的项目中，点击 **Deploy service** 或者 **Add new service** 按钮，并选择 **Deploy your source code**.

![deploy](/framework/deploy.png)

搜索想要部署的 Svelte Kit 代码仓库，并点击导入，你的 Svelte Kit 应用就将自动开始部署。
