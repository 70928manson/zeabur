# 快速开始

从零开始将您的服务部署到 Zeabur 上，只需要以下三个步骤：

### 第一步：注册账号

前往 [登入页面](https://dash.zeabur.com/login) 并且使用 GitHub 帐号进行注册或登录。

![sign in](/sign-in-screenshot.png)

### 第二步：建立新项目

点击创建项目按钮以创建一个新项目。

![new project](/create-project-screenshot.png)

### 第三步：选择要部署的服务

点击刚刚创建好的项目，然后再点击「Add Service」按钮，选择您想要部署的服务。

![deploy service](/deploy-service-screenshot.png)

在部署服务页面中，选择左边的按钮可以从 GitHub 直接将您的源代码自动构建并部署上线。

选择右边的按钮则可以从市场中（例如数据库或开源的工具）中选择一个服务，Zeabur 会自动为您创建一个服务实例。

### 下一步

完成部署以后，您可以了解 [域名绑定](/zh-CN/deploy/domain-binding) 来使用自动带有 TLS 证书的域名对外暴露您的服务，或是了解 [环境变量](/zh-CN/environment/variables) 来了解如何让 Zeabur 自动注入您的程序运行所需要的变量。
