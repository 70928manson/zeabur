# 一鍵部署

使用我們的一鍵部署按鈕，只需點擊一下，就可以將服務部署到 Zeabur。按鈕可以在任何 markdown 文件或 html 文件中使用。它可以將任何 [Zeabur 模板](https://zeabur.com/templates) 轉換為自動在 Zeabur 部署的項目。

![button-demo](/deploy/button/button-demo.png)

## 按鈕的作用

部署按鈕是一個簡單的 html 代碼片段。您可以複製代碼片段並黏貼到任何 markdown 文件或 html 文件中。每個按鈕都將重定向用戶到所需的模板。模板可以是單個服務或由模板作者決定的一組服務。

按鈕也可以以作者的推薦代碼結尾，每個使用推薦代碼註冊並升級到付費計劃的新用戶，作者將獲得 5 美元的積分。

## 複製按鈕

每個按鈕都是與模板相關聯的，只有模板作者才能生成按鈕。

想要複製按鈕，您需要在 [Zeabur Dashboard](https://dash.zeabur.com/account) 中的賬戶頁面，選擇 **模板** 標籤，您可以看到您創建的模板列表。

![created-templates](/deploy/button/created-templates.png)

選擇所需的模板，然後單擊共享按鈕。將彈出一個彈窗，其中包含按鈕代碼。

![popup](/deploy/button/popup.png)

如果您選擇啟用推薦代碼，按鈕代碼將附加推薦代碼，推薦代碼是您在 Zeabur 的用戶名。

## 使用場景

這是將您的項目分享給世界的最佳方式。每個人都可以輕鬆地一鍵部署您的項目。

比如，您可以將按鈕黏貼到以下位置：

- GiHub 的 README 文件
- 個人網站
- 文檔和教程
- 社交媒體簡介

## 前置要求

你需要先創建一個模板才能獲得部署按鈕。您可以在 [Zeabur Dashboard](https://dash.zeabur.com/create-template) 中創建模板。

如下圖所示，您需要填寫模板名稱、描述，並選擇要包含在模板中的服務等等，就像填寫表格一樣來創建你的模板。

![create-template](/deploy/button/create-template.png)

## 賺取返利

就如本文檔開頭所提到的，每個使用您的推薦代碼註冊並升級到付費計劃的新用戶，您將獲得 5 美元的積分。您可以使用積分來支付自己的服務。
